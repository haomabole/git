# Calculations

This directory is for calculation-related code and supporting files.

- `src/`: reusable calculation scripts and modules
- `web/`: browser-based Excel evaluation tool
- `tests/`: tests for calculation code
- `notebooks/`: exploratory notebooks
- `data/`: input data used by calculations
- `results/`: generated outputs and calculation results

Open `web/index.html` in a browser to use the Excel drag-and-drop tool.

The web tool supports CRITIC and entropy objective weights, additive integration,
product integration, and power-product integration.
